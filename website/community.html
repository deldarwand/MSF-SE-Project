The Future of MediDroid

In the future, we hope several features can be added to the app to further enhance its utility and experience:

First, include the possibility to create or edit current POI markers. If a current marker points to a broken bridge on the map, and the user can clearly see the bridge has been fixed, he has the option to edit it. Or, if a broken bridge is discovered, he has the option to create a marker that will be added to the database.
