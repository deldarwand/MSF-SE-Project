Meet the Team

Garrett May

Garrett created the base structure of the Android application for the virtual reality system; He implemented a set of instructions for Bluetooth on Android, built the server for the networking system and read/sent the DHT sensor data. He also created a library for managing packets in C++ and Java/Android.

Hekla Helgadottir

Hekla implemented the gimbal functionality using the Servo library and sourcing the hardware. She handled all client liaisons, as well as designed and presented a poster at UCL’s Digital Health Conference.

Carla Hyenne

Carla implemented the OpenStreetMap view in the Cardboard application, overlaying it with markers within a certain range, as well as tracing the drone’s flight. She also worked on accessing the points of interest from the OpenStreetMap formatted database.

Daniel Eldar

Daniel started by working on the networking environment of the application and continued to the implementation of the main parts of the Android Cardboard application. This includes the landmark marking, and implementation of shaders in GLSL as well as OpenGL programming in Java.
